# 🫧 Bubble To-do

A playful desktop-style sticky note, recreated as a self-contained web demo.

An illustrated girl blows a watercolor bubble; type a to-do in the bubble,
**double-click the girl** to blow it away as a floating bubble, and **click a
floating bubble** to pop it (complete) — with synthesized sound effects.

Live: https://lingkan-wang.github.io/bubble-todo/

- Vanilla HTML/CSS/JS, zero dependencies.
- Animations via CSS + the Web Animations API; sounds via the Web Audio API.
- Originally an Electron desktop widget; this is the in-browser version embedded
  in [my portfolio's Coded Work](https://lingkan-wang.github.io/coded/).
